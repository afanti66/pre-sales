# 需求矩阵（功能需求→技术需求→验收标准）

## F1 组织架构管理

| 功能需求 | 技术需求 | 验收标准 |
|---------|---------|---------|
| 树形组织架构维护 | 组织表设计(parent_id+level+path)，支持递归查询 | 支持10级组织树，增删改查响应≤1s |
| 组织合并/拆分 | 事务性操作，变更记录写入审计表 | 合并/拆分后子节点自动接续，历史可追溯 |
| 组织与人员关联 | user_org关联表，一个人可属于多组织 | 关联变更实时生效 |
| 组织历史追溯 | 组织快照表(org_snapshot)记录每次变更 | 可按时间点查询任意时刻的组织结构 |

## F2 用户生命周期管理

| 功能需求 | 技术需求 | 验收标准 |
|---------|---------|---------|
| 全生命周期管理 | 状态机设计(注册→审核→激活→禁用→删除) | 各状态间转换规则正确，状态变更记录完整 |
| 批量导入导出 | EasyExcel，支持5000行/批次 | 1万用户导入≤30s，支持模板校验 |
| HR系统同步 | SCIM 2.0协议客户端 | 增量同步≤1min，全量同步≤5min |
| 密码策略 | bcrypt(rounds=10) + 复杂度正则校验 | 策略可配置，修改后对非活跃用户不强制 |
| 自助密码重置 | 短信/邮箱验证码验证身份 | 验证码有效期5min，重试3次后锁定 |

## F3 身份认证

| 功能需求 | 技术需求 | 验收标准 |
|---------|---------|---------|
| 用户名/密码认证 | Spring Security + DaoAuthenticationProvider | 认证响应≤500ms |
| 短信验证码 | 集成短信网关SDK | 发送成功率≥99%，到达时间≤10s |
| TOTP动态令牌 | RFC 6238，Google Authenticator兼容 | 30s有效期，可绑定/解绑设备 |
| 数字证书(SM2) | 国密SM2证书链验证 | 支持SM2证书解析和验签 |
| LDAP/AD对接 | Spring Security LDAP | 支持OpenLDAP和Active Directory |
| OAuth2/OIDC/SAML | Spring Security OAuth2 Client | 授权码模式完整流程 |
| CAS协议 | CAS Client集成 | 支持CAS 3.0协议 |

## F4 权限管理

| 功能需求 | 技术需求 | 验收标准 |
|---------|---------|---------|
| RBAC模型 | role_permission关联表，角色-用户多对多 | 权限判定≤100ms |
| ABAC扩展 | SpEL表达式引擎策略定义 | 支持时间、地点、资源属性条件 |
| 角色继承 | 角色树，子角色继承父角色权限 | 权限合并逻辑正确 |
| 角色互斥(SoD) | 互斥角色组定义 | 一个用户不能同时分配互斥角色 |
| 数据权限行级 | MyBatis拦截器注入数据权限SQL | 不同角色看到的数据范围不同 |

## F5 审计日志

| 功能需求 | 技术需求 | 验收标准 |
|---------|---------|---------|
| 全量记录 | AOP切面拦截+MQ异步写入ES | 认证/授权操作100%记录 |
| 日志不可篡改 | 每条日志附带SM3哈希链 | 篡改可检测 |
| 日志检索 | ES查询API | 任意组合条件检索≤3s |
| 异常告警 | 规则引擎+告警通道(短信/邮件) | 5分钟内失败5次触发告警 |

## F6 应用接入

| 功能需求 | 技术需求 | 验收标准 |
|---------|---------|---------|
| REST API | Spring Boot RESTful接口 | 接口响应≤200ms |
| 应用注册 | 应用管理CRUD + 密钥生成 | 密钥支持轮换 |
| 应用监控 | Prometheus + Grafana | 实时TPS/故障率/响应时间可视化 |
