# 某单位统一用户管理与身份认证系统项目 招标技术要求

## 一、项目概述
本项目旨在建设一套统一的用户管理与身份认证系统，实现跨系统的统一用户管理、统一认证、统一权限控制和统一审计。系统需满足等级保护三级要求，支持国产化环境部署。

## 二、总体要求
### 2.1 部署架构
- 支持私有化部署，兼容国产化服务器（鲲鹏/飞腾）+ 国产操作系统（麒麟/UOS）
- 支持集群部署，支持水平扩展，单集群至少支持10万用户
- 接口响应时间：登录认证<200ms，用户查询<500ms，批量操作<3s

### 2.2 安全性要求
- 等保三级合规：日志留存≥180天、三权分立、密码复杂度策略
- 支持国密SM2/SM3/SM4算法
- 防暴力破解：连续失败5次锁定账号，支持滑块验证码
- 会话超时管理：15分钟无操作自动登出

### 2.3 集成要求
- 提供标准RESTful API接口，支持OAuth2.0、SAML2.0、CAS、LDAP协议
- 支持与OA、ERP、邮件系统、企业微信/钉钉集成
- 提供SDK（Java/Python）供业务系统调用

## 三、功能需求
### 3.1 用户管理
1. 用户全生命周期管理：创建/启用/禁用/锁定/解锁/删除
2. 用户信息管理：支持自定义扩展字段，字段类型包括文本/日期/下拉/文件
3. 批量操作：批量导入（Excel/CSV）、批量导出、批量修改属性
4. 组织架构管理：树形组织管理、多层级、组织合并/拆分/迁移
5. 群组管理：动态群组（按规则自动加入）、静态群组（手工维护）
6. 角色管理：支持RBAC0/RBAC1模型，角色继承、角色互斥、角色分级

### 3.2 身份认证
1. 多因素认证：密码+短信验证码、密码+动态令牌、指纹/人脸生物认证
2. 单点登录（SSO）：一次登录访问所有授权系统，支持跨域SSO
3. 认证源管理：支持对接LDAP/AD、企业微信、钉钉、OIDC身份提供商
4. 自助服务：自助密码重置、自助解锁、个人信息修改、手机号/邮箱绑定

### 3.3 权限控制
1. 细粒度权限：功能权限（菜单/按钮）+ 数据权限（组织/个人）+ 字段权限
2. 权限申请与审批：用户自助申请权限→上级审批→自动授权
3. 权限审计：用户权限清单、权限变更日志、闲置权限检测（90天未使用自动回收）

### 3.4 安全审计
1. 操作日志：记录所有用户登录/登出/权限变更/敏感操作
2. 审计报表：登录统计、权限变更统计、异常行为分析
3. 实时告警：异地登录告警、非工作时间登录告警、批量导出告警

## 四、性能指标
1. 并发用户数：≥2000并发
2. 认证响应：≤200ms（P95）
3. 用户查询：≤500ms（百万级用户）
4. 系统可用性：≥99.95%，支持双活/主备切换
5. 全量同步：10万用户同步≤5分钟
