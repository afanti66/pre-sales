# 某企业用户管理系统 · 需求规格

## 1. 业务背景
某中型集团企业（3000+员工）需要替换原有单机版用户管理系统，
建设新一代统一身份认证与权限管理平台。

## 2. 核心需求

### 2.1 身份认证
- 支持 OAuth2.0 + SSO 单点登录
- 多因素认证（MFA）：短信+动态令牌
- LDAP/AD 对接存量目录服务
- 微信/企业微信扫码登录

### 2.2 权限管理
- RBAC 角色权限模型
- 数据级权限（行级+列级）
- 组织树形管理（集团→公司→部门→小组）
- 虚拟组跨组织授权

### 2.3 审计安全
- 全操作审计日志
- 异常登录检测
- 会话管理（强制下线/并发限制）
- 合规保留（日志保存≥180天）

### 2.4 系统要求
- 高可用：双活 + 异地容灾
- 响应：P99 < 500ms
- 并发：支持 5000 同时在线
- 对接：现有 ERP/HR/OA 系统
